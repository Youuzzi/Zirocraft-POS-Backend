# Zirocraft POS - Backend API ☕

Repository ini berisi logika bisnis dan database management untuk **Zirocraft POS**. Dibangun menggunakan framework Spring Boot dengan fokus pada performa dan keamanan data transaksi.

## 🛠️ Tech Stack
- **Java 17**
- **Spring Boot 3.x**
- **Spring Data JPA**
- **MySQL Database**
- **Maven** (Dependency Manager)

## ✨ Fitur Backend (Ongoing)
- **RESTful API** untuk manajemen produk, kategori, dan transaksi.
- **Image Upload System** untuk foto produk.
- **Relasi Database** yang optimal.
- Keamanan konfigurasi menggunakan environment variables (`.env`).
